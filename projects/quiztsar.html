Quiz-Tsar

About Project

My first independent React-Redux project. I decided to make a quiz game that would utilize an API to obtain some trivia quesions. The title is playing on the word "quizzer", and when I thought of "Quiz-Tsar" as being the title, I chose to design the aesthetics of the game around Russian royalty.

The app consists of a simple and easy-to-use interface where the user is guided through the process. After the user begins, they are provided with simple game customization options including number of questions, trivia categories, difficulty and question types.

When a user is playing a quiz, the app utlizes an API from Open Trivia Database to obtain a set of questions based on the user's selection. The app then lists the question one-by-one using buttons on screen for the multiple choice answers. The user is notified of correct or incorrect answers with pop up messages. After all questions are answered the final score is displayed.

My third project overall. What I had to consider in this app was error handling, in case the API was used incorrectly or in case of any other errors. The app uses pop up boxes to give feedback in case of any such incidences. Utilization of the state managment of Redux enabled separation of concerns and could assist scalability of the game for added features in future.

A potential future feature may be the option to play with friends. Another may be to create a game consisting of a number of quizzes where the game tracks and compares your scores between the quizzes you play. This was a feature I thought of in the beginning but chose not to implement at first.
